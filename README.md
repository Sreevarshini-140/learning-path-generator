# 🤖 AI Learning Path Generator

A simple AI-powered web app that generates personalized learning roadmaps based on user input and difficulty level.

---

## 🚀 Live Demo
👉 https://learning-path-generator-project.streamlit.app/

---

## 💡 Features
- Generate learning paths for different domains
- Supports multiple difficulty levels (Beginner, Intermediate, Advanced)
- Clean and interactive UI using Streamlit
- Provides recommended learning resources
- Downloadable learning plan

---

## 🛠 Tech Stack
- Python
- Streamlit

---

## ▶️ How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
